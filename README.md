# WALL_E
## 瓦力智能小管家
### 瓦力偶遇夏娃后
使用最热门的安卓技术，用Picasso、RXvolley、Bmob、Zxing等热门第三方SDK及框架， 开发一个完整的包含有能语音陪聊，能看新闻，能查物流及电话号码归属地，能看美女相册， 能扫描与生成个性化二维码，能定位与查看地图的全新生活类APP
